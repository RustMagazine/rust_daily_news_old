# getrandom crate
一個小巧跨平台，產生random變數的庫
可以產生一個buff長度的亂數

```
fn get_random_buf() -> Result<[u8; 32], getrandom::Error> {
    let mut buf = [0u8; 32];
    getrandom::getrandom(&mut buf)?;
    buf
}
```
